# 🚗 Car Price Prediction using Machine Learning

This project predicts the selling price of used cars using machine learning models.

## Problem Statement
Used car prices depend on multiple factors such as year, fuel type, kilometers driven, and transmission. 
The goal of this project is to build a regression model that can estimate the selling price of a car.

## Dataset
The dataset contains features such as:
- Year
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Prediction

## Models Used
- Linear Regression
- Lasso Regression

## Results
The model was evaluated using the R² score and error metrics.

## Key Learning
- Data preprocessing
- Handling categorical variables
- Regression modeling
- Model evaluation
